Problem Statement

Mental health disorders like anxiety and depression are a pressing global issue, yet a significant number of cases go undiagnosed. Traditional screening methods, such as clinical questionnaires, can be intrusive, time-consuming, and may suffer from response bias.

This leads to a critical gap in early detection. The core problem is the need for a more engaging, implicit, and accessible screening method that can identify potential mental health risks without relying on direct self-reporting, thereby encouraging wider participation and more honest behavioral data.

Motivation

The motivation for this project is to bridge the gap between clinical assessment and public engagement in mental health screening. By transforming the data collection process into a series of engaging mini-games, we aim to:

  • Increase Engagement: Create a non-intimidating experience that users are more likely to complete.
  • Gather Implicit Data: Capture natural behavioral patterns as proxies for mental state.
  • Promote Early Detection: Develop a scalable tool to encourage individuals to seek professional help sooner.
  • Innovate in Digital Health: Apply modern web tech, ML, and AI to a significant healthcare challenge.

Literature Review

Our work builds upon a strong foundation of existing research. The core inspiration is the study by **Vitória et al. (2024)** [1], which successfully used an MLP model to predict anxiety and depression from questionnaire data. Our project innovates by replacing direct questioning with implicit data gathered from gameplay.

The validity of using gameplay data is supported by multiple studies. For instance, **Al-Saggaf et al. (2024)** [2] demonstrated over 93% accuracy in detecting depression and anxiety markers by analyzing game logs. This confirms that behavioral patterns during play can be powerful predictors. Furthermore, a meta-analysis by **Fleming et al. (2017)** [3] showed that "serious games" can have significant positive effects on mental health, validating gamification as an effective methodology.

Objective

The primary objective is to design, develop, and evaluate a novel system for predicting anxiety and depression scores through gamified behavioral data collection.

Specific goals include:

  1. Develop an interactive, multi-game web application ("Mindful Play") that implicitly captures behavioral metrics.
  2. Create a robust data pipeline for collecting and storing gameplay data in Firebase.
  3. Build and train an MLP model to predict mental health scores from the collected gameplay data.
  4. Provide a user-friendly data export function (CSV) for analysis in Google Colab.

Modules Identified

1. User Interface & Gameplay Module

Includes the welcome screen, game hub, and all individual game logic. Responsible for presenting the games and capturing user interactions in real-time.

2. Data Management Module

Handles the collection of gameplay data into a structured object. Manages saving this data to either Firebase Firestore (online) or Local Storage (offline).

3. AI Integration Module

Contains the logic for making API calls to the Gemini API. This includes generating dynamic story content and providing personalized user feedback.

4. Data Export & Analysis Module

Responsible for converting the collected session data into a flattened CSV format for download. This module prepares the data for the final machine learning phase in Google Colab.

Conclusion

This project presents a novel and practical approach to mental health screening by leveraging the engaging nature of games to collect rich behavioral data. By building upon established research that validates the use of machine learning for this purpose, we aim to create a tool that is not only effective but also highly accessible and user-friendly.

The successful completion of this project will result in a functional prototype, a unique dataset of gameplay behavior, and a trained machine learning model capable of predicting anxiety and depression scores. This work has the potential to contribute meaningfully to the field of digital health by offering a new, less intrusive method for early mental health risk detection.
